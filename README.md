# 12306-QueryTool

## **注意：由于 12306 的接口经常变化，本程序代码可能失效。**


### 一、简介
这是一个使用Python3实现的、基于12306网站的火车余票及价格信息查询工具。


### 二、库依赖
+ requests 库: 使用Python访问HTTP资源的必备库
+ prettytable 库: 格式化信息打印工具，能让你像 MySQL 那样打印数据
+ docopt 库: Python3 命令行参数解析工具
+ colorama 库: 命令行着色工具
